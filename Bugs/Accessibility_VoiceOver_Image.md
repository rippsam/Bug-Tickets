## [MTECH] Apply Landing Page Number Images Not Announced

---

### Description
When using VoiceOver the number images on the landing page are not labeled. Currently they are announced as "image". Could this be a regular bulleted list instead of using images?

---

### Replication Steps
1. Go to https://mtec.edu/apply/ and turn on VO
2. Navigate to the number 1 image bullet
3. Observe the announcement of "image"

---

### Expected Behavior
Number 1 bullet is announced as bullet 1 or number 1 or similar

---

### Actual Behavior
Number 1 bullet is announced by VO as "image"

---

### Environment
- **Browser / Device:** Chrome/Mac
- **Test Environment:** Production

---

### Customers Affected
MTECH Students

---

### Category
Accessibility, UI
